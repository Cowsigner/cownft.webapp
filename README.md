Create a `.env` file in the root of your project with the following structure

```

REACT_APP_CANDY_MACHINE_ID=GBWse4e7P9FHTiVNDEu8kKgsnuftNi3YbqpkpcaYBwUQ
```


`npm install`
`npm run start`
